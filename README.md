# ping_pong
A basic 2-player ping-pong game
# Controls : 
* First Player : 
  * W : Up
  * S : Down
  * A : Left
  * D : Right
* Second Player : 
  * Up-Arrow : Up
  * Down-Arrow : Down
  * Left-Arrow : Left
  * Right-Arrow : Right

# Note : 
* Hover mouse on the ball to start the game.
* To restart the game, reload the page, and repeat the above step.
